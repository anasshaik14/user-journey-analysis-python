# user-journey-analysis-python
Analyzing user behavior and conversion patterns using Python, with funnel analysis, drop-off detection, and machine learning prediction.

# User Journey Analysis with Machine Learning

This project analyzes user behavior on a subscription-based learning platform to understand how users navigate, where they drop off, and what leads to conversion (checkout).

## Key Features

- User journey analysis from raw session data
- Conversion funnel analysis
- Drop-off detection across pages
- Visualization dashboard (multiple insights)
- Machine learning model to predict user conversion

## Technologies Used

- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Results

- Identified key drop-off stages in the funnel
- Found most impactful pages for conversion
- Built ML models (Logistic Regression & Random Forest)
- Achieved high predictive performance (AUC score)

## Project Structure

- `user_journey_analysis_final.py` → main analysis script
- `user_journey_raw.csv` → dataset
- `visualization_dashboard.png` → insights dashboard
- `ml_model_results.png` → ML evaluation results
- `executive_summary_final.txt` → business insights summary
